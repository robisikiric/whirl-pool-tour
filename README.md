- node: 20.11.1
- ts-node: 10.9.2
- dependencies
  - @orca-so/whirlpools-sdk: 0.13.0
  - @orca-so/common-sdk: 0.6.0
  - @coral-xyz/anchor: 0.29.0
  - @solana/web3.js: 1.95.0
  - @solana/spl-token: 0.4.1
  - @types/bn.js: 5.1.3
  - bs58: 5.0.0
  - decimal.js: 10.4.3

- Please use `@coral-xyz/anchor` 0.29.0 (Whirlpool is build on Anchor 0.29.0)
